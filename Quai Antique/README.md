# Vue.js
Utilisation du Frameworks Vue pour ce dossier
Marche à suivre pour lancer l'éxécution en local:
    - Ouvrir Gitbash dans le dossier
    - npm init vue@latest
    - Ajouter ou non les options voulues
    - npm install
    - npm run dev 
    - Ctrl + clic gauche sur le lien localhost
